
# dafont-dl

Download fonts from [dafont](https://www.dafont.com/) using the command line

```sh
npx dafont-dl pixel_digivolve
```

## Using the font url

```sh
npx dafont-dl https://www.dafont.com/en/pixel-digivolve.font
```

## Custom output as second argument

```sh
npx dafont-dl pixel_digivolve public/fonts/pixel_digivolve
```


## Custom output as parameter

```sh
npx dafont-dl pixel_digivolve -o public/fonts/pixel_digivolve
